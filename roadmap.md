# Roadmap

## Purpose
This file outlines the next improvements for the NIST control mapping project.

## Current Priorities
- Add more NIST controls and evidence files.
- Expand the control matrix to additional families.
- Create a crosswalk to another framework such as ISO 27001 or CIS Controls.
- Add a simple architecture diagram for the sample system.
- Export the project into a structured format such as OSCAL.

## Immediate Improvements
- Tighten evidence quality for the current controls.
- Add more detailed risk treatment notes where needed.
- Keep the risk register current with likelihood, impact, ownership, and review dates.
- Replace placeholder evidence with more realistic sample artifacts.

## Longer-Term Ideas
- Add a control family summary.
- Add a traceability matrix from control to evidence to risk.
- Add a lessons learned section.
- Add one or two more sample systems to show repeatable mapping.

## Notes
This roadmap is for portfolio development and demonstrates how the project could grow over time.
