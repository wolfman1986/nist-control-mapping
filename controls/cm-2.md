# CM-2 Baseline Configuration

## Control Summary
This control requires the organization to develop, document, and maintain a current baseline configuration for the system.

## Implementation Notes
- Document approved system settings.
- Maintain configuration under change control.
- Review and update the baseline when changes occur.
- Keep version history for previous baselines.
- Separate development, test, and production baselines when needed.

## Evidence
- Baseline configuration document.
- Change management record.
- Configuration export or screenshot.
- Version history or approved change log.
