# AU-2 Event Logging

## Control Summary
This control defines which events the system must log so security activity can be reviewed and investigated.

## Implementation Notes
- Identify important security events.
- Log authentication activity.
- Log privilege or configuration changes.
- Review logs on a regular schedule.
- Keep logs available for investigation and monitoring.

## Evidence
- Logging configuration screenshot or export.
- Sample log entries.
- Log review record.
- SIEM or monitoring output.
