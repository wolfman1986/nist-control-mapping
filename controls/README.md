# Controls

This folder contains individual control writeups for the NIST control mapping project.

## Files
- `ac-2.md` - Account Management.
- `au-2.md` - Event Logging.
- `ia-2.md` - Identification and Authentication.
- `cm-2.md` - Baseline Configuration.

Each file includes a short control summary, implementation notes, evidence examples, and a status label.
