# Evidence

This folder contains example evidence files for the NIST control mapping project.

## Files
- `ac-2.md` - Evidence for Account Management.
- `au-2.md` - Evidence for Event Logging.
- `ia-2.md` - Evidence for Identification and Authentication.
- `cm-2.md` - Evidence for Baseline Configuration.

## Purpose
These files show sample artifacts that support each mapped control and help demonstrate traceability from control to evidence.
