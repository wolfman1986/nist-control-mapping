# Scope

## Purpose
This file defines the boundaries of the NIST control mapping project.

## In Scope
- A single sample information system.
- A limited set of NIST SP 800-53 Rev. 5 controls.
- Control summaries, evidence examples, and gap tracking.
- Sample risk and exception handling.

## Out of Scope
- Full enterprise assessment.
- Formal audit or certification readiness.
- Every NIST control family.
- Live production system evidence.

## Notes
This is a portfolio demonstration meant to show control mapping, evidence traceability, and GRC documentation structure.
